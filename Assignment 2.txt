2. Applications:
    Circular linked lists are used in applications where the entire is accessed one-by-one in a loop.
    1. The browser cache which allows us to hit the BACK button.
    2. Multiplayer games use a circular linked list to swap between players in a loop.
    2. Circular linked lists are helpful in the implementation of stack and queue data structures.
    3. Circular linked lists can be used to manage the computing resources of the computer.
